This project is a Q&A web application similar to Stack Overflow, where users can:
- Post questions (articles)
- Answer questions (comments)
- Comment on answers (sub-comments)
- Vote and accept answers
- View user profiles and activity

Technologies Used
- Backend: Node.js + Express
- Frontend: EJS (Embedded JavaScript Templates)
- Database: MongoDB with Mongoose
- Authentication: express-session
- Styling: Custom CSS

RUN:
  1. git clone https://github.com/BlackHeart3233/Q-A-Web-Application.git
  2. cd Q-A-Web-Application
  3. npm install
  4. Before u start u need to change values for connection to the MongoDB in bin/www
  5. npm start
